# Task Tracking

> Central task list managed by the `todo-manager` skill.

## Current Issue

No active issue. Use `todo-manager init --issue <num> --title "<title>"` to start.

## Task Template

When a new issue is started, this file will be populated with:

```markdown
## Issue #<num>: <title>

### Status: <PLANNING|IN_PROGRESS|TESTING|REVIEW|READY_TO_COMMIT|DONE>

### Tasks
- [ ] Task 1 description
- [ ] Task 2 description

### Blockers
- None

### Notes
-

---
Last updated: <timestamp>
Updated by: <agent-name>
```
